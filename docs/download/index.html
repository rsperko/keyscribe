Download KeyScribe.

Install with Homebrew or download the latest notarized prerelease DMG from GitHub Releases.

Homebrew

Recommended if you already use Homebrew.

brew install rsperko/tap/keyscribe

Direct download

Download the latest `KeyScribe-<version>.dmg`, open it, and drag KeyScribe to Applications.

Open GitHub Releases

Requirements

Current builds are pre-1.0 prereleases.

  • macOS 15+ on Apple silicon.
  • Apple Speech is available only on macOS 26+.
  • Microphone permission is required for speech recognition.
  • Accessibility permission is required for modifier-key triggers and text insertion.
  • In-app updates are not built yet; use Homebrew or download a newer DMG.
